# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn patterns for a subscription-based telecom company. The goal is to identify key factors affecting customer retention and provide actionable insights to reduce churn.

## Dataset
Telco Customer Churn dataset containing information about 7043 customers including demographics, subscription services, and billing details.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning
The dataset was cleaned by converting TotalCharges to numeric format and removing missing values.

## Analysis Performed
- Churn Distribution
- Churn by Contract Type
- Churn by Internet Service
- Monthly Charges vs Churn
- Customer Tenure Analysis

## Key Insights
- Month-to-month contract customers have the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- New customers with shorter tenure are more likely to leave.
- Fiber optic internet users show higher churn rates.

## Business Recommendations
- Encourage long-term contracts through discounts.
- Provide retention offers for high-value customers.
- Improve onboarding experience for new customers.
- Enhance service quality for fiber internet users.
