## Dashboard Section

This dashboard visualizes customer churn patterns and key factors affecting customer retention.

Key visualizations included:

- Customer Churn Distribution
- Churn by Contract Type
- Churn by Internet Service
- Monthly Charges vs Churn
- Customer Tenure Distribution

These visualizations help identify major churn drivers and provide insights for improving customer retention strategies.

The dashboard was created using Python visualization libraries such as Matplotlib and Seaborn.
